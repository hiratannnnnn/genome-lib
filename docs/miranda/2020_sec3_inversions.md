# Sec 3: 転倒数ベースGreedy（符号なし順列）

対象問題：**SbR, SbT, SbRT**（符号なし）

## アルゴリズム（Algorithm 1）

各ステップで「転倒数削減量 / コスト」の比 **ΔInv(π, β) / f(β)** が最大となる λ-再配置 β を選んで適用し続ける。

```
while Inv(π) > 0:
    β = argmax_β ΔInv(π, β) / f(β)   ← M中のλ-再配置から選ぶ
    π ← π · β
```

Inv(π) = 0 ⟺ π = ι（符号なし）なので、必ず終了する（Lemma 5 より常に Inv を減らせる）。

---

## 鍵となる補題

**Lemma 3**（各操作の比の上界）
- λ-逆位 ρ に対して：ΔInv(π, ρ) / |ρ| ≤ (λ−1)/2
- λ-転置 τ に対して：ΔInv(π, τ) / |τ| ≤ λ/4

証明の概略：長さ k の逆位は区間内の転倒 ≤ k(k−1)/2 しか除去できないため比 ≤ (k−1)/2。転置は両区間間の転倒のみ除去できるため比 ≤ k/4。

**Lemma 4**（距離の下界）
- c^r_λ(π) ≥ c^rt_λ(π) ≥ 2·Inv(π) / (λ−1)
- c^t_λ(π) ≥ 4·Inv(π) / λ

証明の概略：Lemma 3 から「転倒数を1減らす最小コストは 2/(λ−1)（逆位）または 4/λ（転置）」が言えるため、Inv(π) 個の転倒をすべて解消するのに少なくともその倍のコストが必要。

**Lemma 5**（アルゴリズムのコスト上界）
- c^t_λ(π) ≤ 2·Inv(π)、c^rt_λ(π) ≤ c^r_λ(π) ≤ 2·Inv(π)

証明の概略：Inv(π) > 0 のとき Lemma 1 より隣接転倒 (πᵢ, πᵢ₊₁) が存在し、τ(i,i+1,i+2) や ρ(i,i+1) はコスト2で転倒数を1減らせる。よって常に「コスト2で転倒1削減」が可能。

---

## 主要定理

**Theorem 1**：λ > 1 のとき、転倒数ベースGreedyは
- SbR と SbRT に対して：**(λ−1)-近似**
- SbT に対して：**(λ/2)-近似**

**証明の流れ**：
- アルゴリズムのコスト ≤ 2·Inv(π)（Lemma 5）
- 最適コスト ≥ 2·Inv(π)/(λ−1)（SbR/SbRT、Lemma 4）
- 最適コスト ≥ 4·Inv(π)/λ（SbT、Lemma 4）
- 比を取ると近似係数が求まる
