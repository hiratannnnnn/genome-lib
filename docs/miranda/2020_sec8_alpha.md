# Sec 8: α > 1 の場合

α が大きいほど長い操作のコストが急増 → **短い操作が相対的に有利** になり近似精度が改善する。

---

## 符号なし順列（SbR, SbT, SbRT）

### Lemma 18（α > 1 での比の上界）

任意の λ-再配置 β に対して：  
ΔInv(π, β) / |β|^α ≤ max_{2 ≤ k ≤ λ} (k−1) / (2k^{α−1})

（α=1 では Lemma 3 の (λ−1)/2 に一致）

### Lemma 19（α > 1 での距離下界）

t = max_{2 ≤ k ≤ λ} (k−1)/(2k^{α−1}) とすると：  
c^r_λ(π) ≥ c^rt_λ(π) ≥ Inv(π)/t　かつ　c^t_λ(π) ≥ Inv(π)/t

### Lemma 20（アルゴリズムのコスト上界）

α > 1 のとき、転倒数ベースGreedy のコスト ≤ 2^α · Inv(π)

---

### α ≥ 2：**2-近似**（Theorem 8）

α ≥ 2 のとき最大値は k = 2 で達成：t = 1/2^{α−1}

- アルゴリズムのコスト ≤ 2^α · Inv(π)
- 最適コスト ≥ 2^{α−1} · Inv(π)
- 近似係数 = 2^α / 2^{α−1} = **2**

### α ≥ 3：**厳密解（多項式時間）**（Theorem 9）

α ≥ 3 のとき k ≥ 4 や k = 3 の項が k = 2 以下になることを示せて t = 1/2^α となり：

- アルゴリズムのコスト ≤ 2^α · Inv(π) = 最適コスト下界
- → アルゴリズムが最適解を出力する（多項式時間で厳密解）

---

## 符号付き順列（SbR̄, SbR̄T）

### Lemma 21（スコア関数 φ の上界）

α > 1 のとき：φ(π, β) ≤ max_{1 ≤ k ≤ λ} k^2/k^α = max_{1 ≤ k ≤ λ} k^{2−α}

**Corollary 1**：α ≥ 2 のとき φ(π, β) ≤ 1

### Lemma 22（ソート列のスコア下界）

α > 1 のとき、Sec 5 のアルゴリズムは φ(π, S) ≥ 1/2^α を満たすソート列 S を生成する

### α ≥ 2：初期近似 2^α-近似（Theorem 10）

Corollary 1（上界 φ ≤ 1）と Lemma 22（下界 φ ≥ 1/2^α）から近似係数 = 2^α

### α ≥ 2：改良して **3-近似**（Theorem 11）

新たな下界を用いることで 3-近似を実現：

c^r̄_λ(π) ≥ c^r̄t_λ(π) ≥ 2^{α−1} · Inv(π)（α ≥ 2 のとき）  
c^r̄_λ(π) ≥ c^r̄t_λ(π) ≥ |E⁻_even| + |E⁺_odd|

アルゴリズム：
1. Inv(π) > 0 の間、長さ2の符号付き逆位で転倒を1つずつ除去（コスト 2^α · Inv(π)）
2. Inv(π) = 0 になったら E⁻_even の各要素に長さ1の逆位を適用（コスト |E⁻_even| + |E⁺_odd|）

合計コスト ≤ 2^α · Inv(π) + |E⁻_even| + |E⁺_odd| ≤ 2·c^r̄_λ(π) + c^r̄_λ(π) = **3·c^r̄_λ(π)**

### α ≥ 3：**厳密解（多項式時間）**

**Lemma 23**：α ≥ 3 のとき、任意の符号付き λ-逆位 ρ̄ を super short 逆位列で置換でき、そのコスト ≤ |ρ̄|^α。

証明の核心（長さ k の逆位の置換コスト）：
- 各要素を正しい位置へ移動するのに長さ2の逆位を計 k(k−1)/2 本使う
- 符号修正に長さ1の逆位を k 本使う
- 合計コスト ≤ 2^{α−1}·k(k−1) + k ≤ k^α（α ≥ 3 のとき成立）

**Lemma 24**：α ≥ 3 のとき、任意の λ-転置 τ も super short 操作列で置換できコスト ≤ |τ|^α。

**Theorem 12**（SbR̄、多項式時間）：  
α ≥ 3 のとき、最適解は super short 操作のみからなる。  
最適コスト = 2^α · Inv(π) + |E⁻_even| + |E⁺_odd|（下界 = 上界）  
→ 多項式時間で厳密解

**Theorem 13**（SbR̄T、多項式時間）：  
α ≥ 3 のとき同様に、最適コスト = 2^α · Inv(π) + codd(π)  
補助補題：
- **Lemma 25**：長さ2の操作で転倒を除去しても codd(π) は減少しない
- **Lemma 26**：Inv(π) > 0 のとき、転倒を除去しつつ codd(π) を変化させない長さ2の操作が必ず存在する
- **Lemma 27**：c^r̄t_2(π) ≤ codd(π) + 2^α · Inv(π)（super short 操作のみでこのコストを達成）

---

## α > 1 の結果まとめ

| α の条件 | 符号なし（SbR/SbT/SbRT） | 符号付き（SbR̄/SbR̄T） |
|---------|----------------------|---------------------|
| α ≥ 2 | **2-近似** | **3-近似** |
| α ≥ 3 | **多項式時間・厳密解** | **多項式時間・厳密解** |
