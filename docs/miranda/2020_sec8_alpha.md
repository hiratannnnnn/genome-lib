# Sec 8: α > 1 の場合

α が大きいほど長い操作のコストが急増し、**短い操作が相対的に有利** になるため近似精度が改善する。

---

## 前提定義

### 順列と単位順列

長さ n の符号なし／符号付き順列の定義と単位順列 ι は Sec 3 / Sec 5 と同様。

### 再配置の種類と長さ

逆位・転置・λ-再配置・super short の定義は Sec 3 / Sec 5 と同様。

### コスト関数（一般の α）

本節では α ≥ 1 を実数パラメータとして、再配置 β のコストを **f(β) = |β|^α** と定義する。  
α = 1 のとき Sec 3–7 の設定に一致する。操作列 S の総コストは f(S) = Σ f(βᵢ)。

### λ-再配置距離

λ-再配置のみを使って π をソートする最小総コスト（コスト関数 f(β) = |β|^α）を **λ-再配置距離** と呼ぶ。  
記法 cʳλ, cᵗλ, cʳᵗλ, cr̄λ, cr̄tλ は Sec 3 / Sec 5 と同様。

### 転倒数と ΔInv

転倒数 Inv(π)、削減量 ΔInv(π, β) = Inv(π) − Inv(π·β) の定義は Sec 3 と同様（符号付きでは絶対値で比較）。

### E⁻_even, E⁺_odd（符号付き順列用）

- **E⁻_even(π)** = { πᵢ : πᵢ < 0 かつ ent(πᵢ) が偶数 }
- **E⁺_odd(π)** = { πᵢ : πᵢ > 0 かつ ent(πᵢ) が奇数 }

ここで ent(πᵢ) = ||πᵢ| − i| はエントロピー（Sec 4 参照）。

### スコア関数 φ（符号付き順列用）

φ(π, S) = [(2·Inv(π) + |E⁻_even(π)| + |E⁺_odd(π)|) − (2·Inv(π·S) + |E⁻_even(π·S)| + |E⁺_odd(π·S)|)] / f(S)

### 奇連結成分数 codd（符号付き順列用）

転倒グラフ G(π) の奇連結成分数 codd(π)（Sec 7 参照）。

---

## 符号なし順列（SbR, SbT, SbRT）

### Lemma 18（α > 1 での比の上界）

任意の符号なし順列 π、整数 λ > 1、実数 α > 1 に対して、任意の λ-再配置 β は：

ΔInv(π, β) / |β|^α ≤ max_{2 ≤ k ≤ λ} (k−1) / (2k^{α−1})

α = 1 のとき右辺は (λ−1)/2 となり Lemma 3 に一致する。

---

### Lemma 19（α > 1 での距離下界）

t = max_{2 ≤ k ≤ λ} (k−1)/(2k^{α−1}) とすると：  
cʳλ(π) ≥ cʳᵗλ(π) ≥ Inv(π)/t　かつ　cᵗλ(π) ≥ Inv(π)/t

証明：Lemma 18 の上界から「転倒数を1減らす最小コストは 1/t」が言えるため、下界 Inv(π)/t が成立する。

---

### Lemma 20（α > 1 でのアルゴリズムのコスト上界）

α > 1 のとき、Sec 3 の転倒数ベースGreedy（Algorithm 1）は総コスト ≤ 2^α · Inv(π) で π をソートする。

証明の概略：Inv(π) > 0 のとき隣接転倒 (πᵢ, πᵢ₊₁) が存在し、τ(i,i+1,i+2) または ρ(i,i+1) はコスト 2^α で転倒数を1減らす（長さ2の操作のコストが 2^α）。

---

### α ≥ 2：2-近似（Theorem 8）

α ≥ 2 のとき t の最大値は k = 2 で達成され、t = 1/2^{α−1} となる：

| | SbR / SbT / SbRT |
|---|---|
| アルゴリズムのコスト上界（Lemma 20） | ≤ 2^α · Inv(π) |
| 最適コストの下界（Lemma 19） | ≥ 2^{α−1} · Inv(π) |
| 近似係数 | 2^α / 2^{α−1} = **2** |

**Theorem 8**：α ≥ 2 かつ λ > 1 のとき、転倒数ベースGreedy は SbR, SbT, SbRT に対して **2-近似**。

---

### α ≥ 3：厳密解・多項式時間（Theorem 9）

α ≥ 3 のとき、k ≥ 3 の項がすべて k = 2 以下になることを示せて t = 1/2^α となる：

| | SbR / SbT / SbRT |
|---|---|
| アルゴリズムのコスト上界（Lemma 20） | = 2^α · Inv(π) |
| 最適コストの下界（Lemma 19） | ≥ 2^α · Inv(π) |

上界 = 下界なので **アルゴリズムが最適解を出力する**。  
**Theorem 9**：α ≥ 3 かつ λ > 1 のとき、SbR, SbT, SbRT は多項式時間で厳密解が求まる。

---

## 符号付き順列（SbR̄, SbR̄T）

### Lemma 21（スコア関数 φ の上界）

任意の符号付き順列 π、λ > 1、α > 1 に対して任意の λ-再配置 β は：

φ(π, β) ≤ max_{1 ≤ k ≤ λ} k² / k^α = max_{1 ≤ k ≤ λ} k^{2−α}

**Corollary 1**：α ≥ 2 のとき φ(π, β) ≤ 1（k = 1 で最大値 1^{2−α} = 1）。

---

### Lemma 22（ソート列のスコア下界）

α > 1 のとき、Sec 5 の Algorithm 3 は φ(π, S) ≥ 1/2^α を満たすソート列 S を生成する。  
（長さ2の逆位で隣接転倒を除去すると φ = 2/2^α = 1/2^{α−1} ≥ 1/2^α）

---

### α ≥ 2：初期近似 2^α-近似（Theorem 10）

Corollary 1（φ ≤ 1）と Lemma 22（φ ≥ 1/2^α）から近似係数 = 1/(1/2^α) = 2^α。  
**Theorem 10**：α ≥ 2 のとき SbR̄ と SbR̄T に対して **2^α-近似**。

---

### α ≥ 2：改良して **3-近似**（Theorem 11）

新たな下界を用いることで 2^α-近似から 3-近似に改善する。

**下界**（α ≥ 2 のとき）：
- cr̄λ(π) ≥ cr̄tλ(π) ≥ 2^{α−1} · Inv(π)
- cr̄λ(π) ≥ cr̄tλ(π) ≥ |E⁻_even(π)| + |E⁺_odd(π)|

**アルゴリズム**：
1. Inv(π) > 0 の間、長さ2の符号付き逆位で転倒を1つずつ除去（各ステップのコスト 2^α）。
2. Inv(π) = 0 になったら E⁻_even の各要素に長さ1の逆位を適用して符号を修正（各ステップのコスト 1）。

**合計コスト** ≤ 2^α · Inv(π) + |E⁻_even| + |E⁺_odd| ≤ 2·cr̄λ(π) + cr̄λ(π) = **3·cr̄λ(π)**

**Theorem 11**：α ≥ 2 かつ λ > 1 のとき SbR̄ と SbR̄T に対して **3-近似**。

---

### α ≥ 3：厳密解・多項式時間（Theorems 12–13）

**Lemma 23**：α ≥ 3 のとき、任意の符号付き λ-逆位 ρ̄ を super short 逆位列で模倣でき、そのコスト ≤ |ρ̄|^α。

証明の核心（長さ k の逆位の模倣コスト）：
- 各要素を正しい位置へ移動するのに長さ2の逆位を計 k(k−1)/2 回使う。コスト 2^α · k(k−1)/2。
- 符号修正に長さ1の逆位を k 回使う。コスト k。
- 合計 ≤ 2^{α−1}·k(k−1) + k ≤ k^α（α ≥ 3 のとき k=3 と k≥4 で別々に確認）。

**Lemma 24**：α ≥ 3 のとき、任意の λ-転置 τ も super short 操作列で模倣でき、コスト ≤ |τ|^α。

**Lemma 25**：長さ2の操作で転倒を除去しても codd(π) は減少しない。

証明の概略：除去される辺 e = (πᵢ, πᵢ₊₁) について (i) e がカット辺の場合：e を含む連結成分が2つに分かれるが奇連結成分数は変化しないか増加する。(ii) e がカット辺でない場合：連結成分の頂点集合は変わらず codd(π) も変化しない。

**Lemma 26**：Inv(π) > 0 のとき、ΔInv = 1 かつ codd(π) 不変となる長さ2の操作が必ず存在する。

証明の概略：隣接転倒 (πᵢ, πᵢ₊₁) を含む辺 e のカット辺か否かと両側連結成分の奇偶の組み合わせに応じて、転置または符号付き逆位を使い分けることで codd(π) を変化させずに転倒を除去できる。

**Lemma 27**：任意の符号付き順列 π と α ≥ 1 に対して cr̄t₂(π) ≤ codd(π) + 2^α · Inv(π)。

証明の概略：(1) Lemma 26 を用いて Inv(π) 回の長さ2操作で転倒を全除去（コスト 2^α · Inv(π)）。(2) G(π) の各奇連結成分に長さ1の逆位を1回適用（コスト codd(π)）。

**Theorem 12**（SbR̄、多項式時間）：  
α ≥ 3 のとき、Lemmas 23, 24 より最適解は super short 操作のみで構成できる。  
最適コストの下界と上界がともに 2^α · Inv(π) + |E⁻_even| + |E⁺_odd| になることを示せるため、この値を達成するアルゴリズムが最適解を与える（多項式時間で厳密解）。

**Theorem 13**（SbR̄T、多項式時間）：  
α ≥ 3 のとき、同様に最適コスト = 2^α · Inv(π) + codd(π)。  
Lemma 27 のアルゴリズムがこの値を達成し、Lemmas 25, 26 を用いた下界証明と一致するため多項式時間で厳密解が求まる。

---

## α > 1 の結果まとめ

| α の条件 | 符号なし（SbR / SbT / SbRT） | 符号付き（SbR̄ / SbR̄T） |
|---------|------------------------------|------------------------|
| α ≥ 2 | **2-近似**（Theorem 8） | **3-近似**（Theorem 11） |
| α ≥ 3 | **多項式時間・厳密解**（Theorem 9） | **多項式時間・厳密解**（Theorems 12–13） |
