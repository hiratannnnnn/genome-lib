# Sec 5: スコア関数 φ ベースのアルゴリズム（符号付き順列）

対象問題：**SbR̄, SbR̄T**（符号付き）

## スコア関数 φ の定義

符号付き順列では転倒数だけでなく「負の要素（符号の不一致）」も考慮する必要がある。  
スコア関数 φ は転倒数と E⁻_even・E⁺_odd の変化をコストで割ったもの：

```
φ(π, S) = [(2·Inv(π) + |E⁻_even(π)| + |E⁺_odd(π)|) − (2·Inv(π·S) + |E⁻_even(π·S)| + |E⁺_odd(π·S)|)] / f(S)
```

- 恒等順列 ι は **Inv(ι) = |E⁻_even(ι)| = |E⁺_odd(ι)| = 0** を満たす唯一の順列
- φ が大きいほど「コストに対してソートが進んでいる」

## アルゴリズム（Algorithm 3）

```
while π ≠ ι:
    β = argmax_β φ(π, β)    ← M中のλ-再配置から選ぶ
    π ← π · β
```

---

## 鍵となる補題

**Lemma 10**：任意の符号付き λ-逆位 ρ̄ に対して φ(π, ρ̄) ≤ λ

証明の概略：長さ k の逆位の転倒数変化 ≤ k(k−1)/2 と |E⁻_even|+|E⁺_odd| の変化 ≤ k を合わせると分子 ≤ k² となり、コスト k で割ると ≤ k ≤ λ。

**Lemma 11**：任意の λ-転置 τ に対しても φ(π, τ) ≤ λ（Lemma 10 と同様の証明）

**Lemma 12**：π ≠ ι のとき、φ(π, ρ̄) ≥ 1 となる super short 符号付き逆位 ρ̄ が必ず存在する。

証明の概略（2ケース）：
1. Inv(π) = 0 のとき：π ≠ ι だから E⁻_even に負要素 πᵢ が存在し、ρ̄(i,i) で符号を直すと φ = 1。
2. Inv(π) > 0 のとき：Lemma 1 より隣接転倒 (πᵢ, πᵢ₊₁) が存在し、ρ̄(i,i+1) で転倒を1除去（Lemma 2 より |E⁻_even|+|E⁺_odd| は変化しない）→ φ = 2/2 = 1。

---

## 主要定理

**Theorem 3**：λ > 1 のとき、Algorithm 3 は SbR̄ と SbR̄T に対して **λ-近似**。

**証明の流れ**：
- Lemma 12 より毎ステップ φ ≥ 1 を達成できる → ソート列が存在し、アルゴリズムは停止する
- Lemma 10, 11 より各操作の φ ≤ λ → コスト上界 ≤ λ × 最適コスト下界
- 下界：最適コストは初期ポテンシャル (2·Inv(π) + |E⁻_even| + |E⁺_odd|) / λ 以上
