# Sec 5: スコア関数 φ ベースのアルゴリズム（符号付き順列）

対象問題：**SbR̄, SbR̄T**（符号付き）

---

## 前提定義

### 符号付き順列と単位順列

長さ n の **符号付き順列** π = (π₁ π₂ … πₙ) は πᵢ ∈ {−n, …, −1, +1, …, +n} かつ |πᵢ| がすべて異なる並び替えである。  
**単位順列** ι = (+1 +2 … +n)（全要素が正）がソートの目標となる。

### 再配置の種類・長さ・コスト

**符号付き逆位** ρ̄(i, j)（1 ≤ i ≤ j ≤ n）は区間 [i, j] の要素を逆順にし、各要素の符号を反転する：

(π₁ … πᵢ₋₁  πᵢ … πⱼ  πⱼ₊₁ … πₙ) · ρ̄(i,j) = (π₁ … πᵢ₋₁  −πⱼ … −πᵢ  πⱼ₊₁ … πₙ)

長さ |ρ̄(i,j)| = j−i+1。長さ1の逆位 ρ̄(i,i) は要素の符号のみを反転する。

**転置** τ(i, j, k)（1 ≤ i < j < k ≤ n+1）は符号なしと同様に2区間を交換する（符号は変化しない）。長さ |τ(i,j,k)| = k−i。

整数 λ > 1 に対し |β| ≤ λ を満たす再配置を **λ-再配置** と呼ぶ。  
なお |β| ≤ 2 を満たす再配置を **super short** と呼ぶ。

コスト関数は **f(β) = |β|**（本節では α = 1）。操作列 S の総コストは f(S) = Σ f(βᵢ)。

### λ-再配置距離

π を λ-再配置のみで ι にソートする最小総コストを **λ-再配置距離** と呼び、次の記法を使う：

- cr̄λ(π)：SbR̄（符号付き λ-逆位のみ）
- cr̄tλ(π)：SbR̄T（符号付き λ-逆位と λ-転置）

### 転倒数

符号付き順列における要素の対 (πᵢ, πⱼ) が **転倒** であるとは、i < j かつ |πᵢ| > |πⱼ| が成り立つことをいう（絶対値で比較）。  
**転倒数** Inv(π) を転倒の総数とする。

### エントロピー

要素 πᵢ の **エントロピー** を ent(πᵢ) = ||πᵢ| − i| と定義する。

### E⁻_even と E⁺_odd

- **E⁻_even(π)** = { πᵢ : πᵢ < 0 かつ ent(πᵢ) が偶数 }
- **E⁺_odd(π)** = { πᵢ : πᵢ > 0 かつ ent(πᵢ) が奇数 }

これらの集合は「符号の向きがおかしい要素」を捉えるために導入される。単位順列 ι は Inv(ι) = |E⁻_even(ι)| = |E⁺_odd(ι)| = 0 を満たす唯一の符号付き順列である。

**Lemma 2**（Galvão et al. 2015）：任意の符号付き逆位 ρ̄ で |ρ̄| = 2 のとき、π' = π·ρ̄ に対して |E⁻_even(π)| + |E⁺_odd(π)| = |E⁻_even(π')| + |E⁺_odd(π')| が成り立つ。

### スコア関数 φ

順列 π と再配置列 S に対し、**スコア関数** φ を次で定義する：

φ(π, S) = [(2·Inv(π) + |E⁻_even(π)| + |E⁺_odd(π)|) − (2·Inv(π·S) + |E⁻_even(π·S)| + |E⁺_odd(π·S)|)] / f(S)

φ が大きいほど「コストに対してソートが進んでいる」ことを意味する。

---

## アルゴリズム（Algorithm 3）

各ステップで φ が最大となる λ-再配置を貪欲に選んで適用する。

```
入力：符号付き順列 π、整数 λ > 1、λ-再配置のみからなるモデル M
出力：π をソートする M の操作列

while π ≠ ι:
    β = argmax{ φ(π, β) : β ∈ M }
    π ← π · β
    S に β を追加
return S
```

ι は Inv + |E⁻_even| + |E⁺_odd| = 0 を満たす唯一の順列なので、これが0になれば終了する。  
Lemma 12 より常に φ ≥ 1 の操作が存在するため、アルゴリズムは必ず終了する。

---

## 鍵となる補題

**Lemma 10**（φ の上界：逆位）

任意の符号付き順列 π と符号付き λ-逆位 ρ̄ に対して φ(π, ρ̄) ≤ λ。

証明の概略：長さ k の逆位の転倒数変化 ≤ k(k−1)/2 なので 2ΔInv ≤ k(k−1)。また k 要素が影響を受け E⁻_even ∩ E⁺_odd = ∅ なので |E⁻_even|+|E⁺_odd| の変化 ≤ k。合計すると分子 ≤ k(k−1)+k = k²、コスト k で割ると φ ≤ k ≤ λ。

---

**Lemma 11**（φ の上界：転置）

任意の符号付き順列 π と λ-転置 τ に対して φ(π, τ) ≤ λ（Lemma 10 と同様の証明）。

---

**Lemma 12**（φ ≥ 1 の操作の存在）

符号付き順列 π ≠ ι のとき、φ(π, ρ̄) ≥ 1 となる super short 符号付き逆位 ρ̄ が必ず存在する。

証明の概略（2ケース）：
1. **Inv(π) = 0 のとき**：π ≠ ι なので E⁻_even に負要素 πᵢ が存在し、ρ̄(i,i) で符号を直すと φ = 1/1 = 1。
2. **Inv(π) > 0 のとき**：隣接転倒 (πᵢ, πᵢ₊₁) が存在し（符号なし版の Lemma 1 と同様）、ρ̄(i,i+1) でその転倒を除去すると ΔInv = 1 かつ Lemma 2 より E⁻_even+E⁺_odd の変化 = 0。よって φ = 2/2 = 1。

---

## 主要定理

**Theorem 3**：λ > 1 のとき、Algorithm 3 は SbR̄ と SbR̄T に対して **λ-近似**。

**証明の流れ**：

| | SbR̄ / SbR̄T |
|---|---|
| 各操作の φ 上界（Lemma 10, 11） | φ(π, β) ≤ λ |
| 常に φ ≥ 1 の操作が存在（Lemma 12） | アルゴリズムは停止し、コスト上界 ≤ λ × (最適コスト下界) |
| 最適コストの下界 | ≥ (2·Inv(π) + \|E⁻_even\| + \|E⁺_odd\|) / λ |
| 近似係数 | **λ** |
