# Sec 4: エントロピーベースGreedy（符号なし順列）

対象問題：**SbR, SbT, SbRT**（符号なし）

---

## 動機

Sec 3 の転倒数ベースでは SbT に λ/2-近似しか得られなかった。  
エントロピーを用いると SbR, SbT, SbRT すべてで **λ-近似** が統一的に得られる。

---

## 前提定義

### 順列と単位順列

長さ n の **符号なし順列** π = (π₁ π₂ … πₙ) は {1, 2, …, n} の並び替えである。
**単位順列** ι = (1 2 … n) がソートの目標となる。

### 再配置の種類・長さ・コスト

**逆位** ρ(i, j)（1 ≤ i < j ≤ n）は区間 [i, j] の要素を逆順にする。長さ |ρ(i,j)| = j−i+1。  
**転置** τ(i, j, k)（1 ≤ i < j < k ≤ n+1）は隣接する2区間 [i,j) と [j,k) を交換する。長さ |τ(i,j,k)| = k−i。

整数 λ > 1 に対し |β| ≤ λ を満たす再配置を **λ-再配置** と呼ぶ。  
**再配置モデル** M は使用できる λ-再配置の集合（逆位のみ・転置のみ・両方など）。

コスト関数は **f(β) = |β|**（本節では α = 1）。操作列 S の総コストは f(S) = Σ f(βᵢ)。  
なお |β| ≤ 2 を満たす再配置を **super short** と呼ぶ。

### λ-再配置距離

π をモデル M の λ-再配置のみで ι にソートする最小総コストを **λ-再配置距離** と呼び、次の記法を使う：

- cʳλ(π)：SbR（λ-逆位のみ）
- cᵗλ(π)：SbT（λ-転置のみ）
- cʳᵗλ(π)：SbRT（λ-逆位と λ-転置）

### エントロピー

要素 πᵢ の **エントロピー** を「現在位置 i とあるべき位置 |πᵢ| のズレ」として定義する：

ent(πᵢ) = ||πᵢ| − i|

順列 π の **エントロピー** を各要素のエントロピーの総和とする：

ent(π) = Σᵢ₌₁ⁿ ent(πᵢ)

**基本性質**：
- ent(π) = 0 ⟺ π = ι（符号なし順列においては単位順列のみ）

**例**：π = (4 5 2 3 1) では ent(π₁)=3, ent(π₂)=3, ent(π₃)=1, ent(π₄)=1, ent(π₅)=4 なので ent(π) = 12。

また、再配置 β による **エントロピー削減量** を Δent(π, β) = ent(π) − ent(π·β) と表す。

---

## アルゴリズム（Algorithm 2）

コスト1あたりに最もエントロピーを削減できる λ-再配置を選ぶ貪欲法。ただし最良操作でも削減効率が 1/2 未満になる場合は Lemma 9 のフォールバック手順を使う。

```
入力：符号なし順列 π、整数 λ > 1、λ-再配置のみからなるモデル M
出力：π をソートする M の操作列

while π ≠ ι:
    β = argmax{ Δent(π, β) / f(β) : β ∈ M }
    if Δent(π, β) / f(β) ≥ 1/2:
        π ← π · β
        S に β を追加
    else:
        Lemma 9 の手順で πᵢ と πⱼ（|πᵢ| ≥ j かつ |πⱼ| ≤ i）を交換
        S に操作列を追加
return S
```

ent(π) = 0 ⟺ π = ι（符号なし）なので、エントロピーが0になれば終了する。  
Lemma 9 より常にエントロピーを削減する操作が存在するため、アルゴリズムは必ず終了する。

---

## 鍵となる補題

**Lemma 6**（各操作のエントロピー比の上界）

任意の順列 π と λ-再配置 β に対して：Δent(π, β) / |β| ≤ λ/2

証明の概略：長さ k の逆位では各要素の移動量の合計が最大 k²/2 なので Δent ≤ k²/2、比 ≤ k/2 ≤ λ/2。転置 τ(i,j,k) では両区間の移動量合計が 2(k−j)(j−i) ≤ 2(k/2)² = k²/2 なので同様に比 ≤ k/2 ≤ λ/2。

---

**Lemma 7**（距離の下界）

任意の符号なし順列 π と λ > 1 に対して：  
cʳλ(π) ≥ cʳᵗλ(π) ≥ 2·ent(π)/λ　かつ　cᵗλ(π) ≥ 2·ent(π)/λ

証明の概略：Lemma 6 より「エントロピーを1減らす最小コストは 2/λ」。ent(ι) = 0 なので任意のソート列は ent(π) 分のエントロピーを削減しなければならず、下界 2·ent(π)/λ が成り立つ。

---

**Lemma 8**（交換可能な要素対の存在）

ent(π) > 0 ならば、|πᵢ| ≥ j かつ |πⱼ| ≤ i となる組 (i, j)（i < j）が必ず存在する。

証明の概略：ent(π) > 0 より定位置にない要素が存在する。最大絶対値を持つ定位置外の要素 πᵢ（|πᵢ| > i の最右端）を取り、その右側で最初に |πⱼ| < j となる πⱼ を取ると条件を満たす。

---

**Lemma 9**（フォールバックの保証）

ent(π) > 0 ならば、Δent(π, S) / f(S) ≥ 1/2 となる super short 再配置列 S が存在する。

証明の概略：Lemma 8 の πᵢ と πⱼ を交換すると ent が 2(j−i) 減少する。この交換は super short 逆位を 2(j−i)−1 回使う列（総コスト ≤ 4(j−i)）で実現できるため、比 ≥ 2(j−i) / 4(j−i) = 1/2。

---

## 主要定理

**Theorem 2**：λ > 1 のとき、Algorithm 2 は SbR, SbT, SbRT に対して **λ-近似**。

**証明の流れ**：

| | SbR / SbT / SbRT |
|---|---|
| アルゴリズムのコスト上界（Lemma 9） | ≤ 2·ent(π) |
| 最適コストの下界（Lemma 7） | ≥ 2·ent(π)/λ |
| 近似係数 | 2·ent(π) / [2·ent(π)/λ] = **λ** |
