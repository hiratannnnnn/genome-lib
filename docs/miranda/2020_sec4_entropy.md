# Sec 4: エントロピーベースGreedy（符号なし順列）

対象問題：**SbR, SbT, SbRT**（符号なし）

## 動機

Sec 3 の転倒数ベースでは SbT に λ/2-近似しか得られなかった。  
エントロピーを用いると SbR, SbT, SbRT すべてで **λ-近似** が得られる。  
（SbR/SbRT では Sec 3 の方が良いが、SbT では同等、統一的な枠組みとして有用）

## アルゴリズム（Algorithm 2）

各ステップで「エントロピー削減量 / コスト」の比 **Δent(π, β) / f(β)** が最大となる λ-再配置 β を選ぶ。ただし比が 1/2 未満のときは Lemma 9 のフォールバック手順を使う。

```
while π ≠ ι:
    β = argmax_β Δent(π, β) / f(β)
    if Δent(π, β) / f(β) ≥ 1/2:
        π ← π · β
    else:
        Lemma 9 の手順で πᵢ と πⱼ を交換する（比 ≥ 1/2 を保証）
```

ent(π) = 0 ⟺ π = ι（符号なし）なので、エントロピーを下げ続ければ必ず終了する。

---

## 鍵となる補題

**Lemma 6**（各操作のエントロピー比の上界）  
任意の λ-再配置 β に対して：Δent(π, β) / |β| ≤ λ/2

証明の概略：長さ k の逆位は各要素を最大 k 移動させ、総エントロピー変化 ≤ k²/2。比は k/2 ≤ λ/2。転置についても同様（2区間の積が最大 (k/2)²）。

**Lemma 7**（距離の下界）  
c^r_λ(π) ≥ c^rt_λ(π) ≥ 2·ent(π)/λ　かつ　c^t_λ(π) ≥ 2·ent(π)/λ

証明の概略：Lemma 6 より「エントロピーを1減らす最小コスト = 2/λ」。ソートには ent(π) 分のエントロピー削減が必要なので下界は 2·ent(π)/λ。

**Lemma 8**：ent(π) > 0 ならば、|πᵢ| ≥ j かつ |πⱼ| ≤ i となる i < j の組 (πᵢ, πⱼ) が必ず存在する。

（証明は最大値の πᵢ と最初に条件を満たす πⱼ を取る構成的な議論）

**Lemma 9**：ent(π) > 0 ならば、Δent(π, S) / f(S) ≥ 1/2 となる super short 再配置列 S が存在する。

証明の概略：Lemma 8 の πᵢ と πⱼ を交換するとエントロピーが 2(j−i) 減る。この交換を super short 逆位列（コスト ≤ 4(j−i)）で実現できるため比 ≥ 1/2。

---

## 主要定理

**Theorem 2**：λ > 1 のとき、エントロピーベースGreedyは SbR, SbT, SbRT に対して **λ-近似**。

**証明の流れ**：
- Lemma 9 より f(S) ≤ 2·ent(π)（アルゴリズムのコスト上界）
- Lemma 7 より最適コスト ≥ 2·ent(π)/λ
- 比を取ると近似係数 = λ
