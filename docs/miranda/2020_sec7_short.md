# Sec 7: 短い再配置の場合（λ = 3）

Sec 3–5 のアルゴリズムを **λ = 3** に特化して解析・改良する。

---

## 前提定義

### 順列と単位順列

長さ n の符号なし／符号付き順列の定義と単位順列 ι は Sec 3 / Sec 5 と同様。

### 再配置の長さ区分

再配置 β の長さ |β| に応じて次の呼び方をする：
- **super short**：|β| ≤ 2
- **short**：|β| ≤ 3

本節では λ = 3 の場合を扱うため、使用できる操作は short 再配置のみ。

### コスト関数

f(β) = |β|（α = 1）。操作列 S の総コストは f(S) = Σ f(βᵢ)。

### 転倒数

要素の対 (πᵢ, πⱼ) が **転倒** であるとは、i < j かつ |πᵢ| > |πⱼ| が成り立つことをいう（符号付き・なしとも絶対値で比較）。**転倒数** Inv(π) を転倒の総数とする。

### 転倒グラフと奇連結成分数

符号付き順列 π の **転倒グラフ（inversion graph）** G(π) = (V, E) を次で定める：
- 頂点集合 V = {π₁, …, πₙ}
- 辺集合 E = { (πᵢ, πⱼ) : (πᵢ, πⱼ) は π の転倒 }

連結成分 C が **奇連結成分** であるとは、C に含まれる負の要素（πᵢ < 0）の個数が奇数であることをいう。
- c(π)：G(π) の連結成分数
- **codd(π)**：G(π) の奇連結成分数

G(π) の辺 e で、その削除により連結成分数が増加するものを **カット辺（cut-edge）** と呼ぶ。

### スコア関数 ψ（λ = 3 専用）

符号付き順列 π と再配置列 S に対して、**スコア関数** ψ を次で定義する：

ψ(π, S) = [(2·Inv(π) + codd(π)) − (2·Inv(π·S) + codd(π·S))] / f(S)

ι は Inv(ι) + codd(ι) = 0 を満たす唯一の符号付き順列である。

---

## 符号なし順列の結果

### SbR と SbRT：Theorem 1 より直接 **2-近似**

λ = 3 のとき λ−1 = 2 なので、Sec 3 の転倒数ベースGreedy（Algorithm 1）がそのまま 2-近似になる。

### SbT：Lemma 15 で改良して **4/3-近似**

Theorem 1 では λ/2 = 3/2 の近似係数だが、λ = 3 の転置に対してより精密な下界を示すことで改善できる。

**Lemma 15**：転倒数ベースGreedy（Algorithm 1）は SbT（λ = 3）に対して **4/3-近似**。

証明の概略：
- 短い転置 τ(i,j,k) で長さ k ≤ 3 のとき、ΔInv(π,τ)/|τ| ≤ (j−i)(k−j)/k ≤ 2/3（k=3 のとき最大は (j−i)=1,(k−j)=2 のとき 2/3）。
- よって cᵗ₃(π) ≥ (3/2)·Inv(π)（Lemma 4 よりも強い下界）。
- アルゴリズムのコスト ≤ 2·Inv(π)（Lemma 5）。
- 近似係数 = 2·Inv(π) / [(3/2)·Inv(π)] = **4/3**。

---

## 符号付き順列の結果

### SbR̄, SbR̄T：Theorem 3 より **3-近似**

λ = 3 のとき Sec 5 の Algorithm 3 がそのまま 3-近似（= λ-近似）になる。

### SbR̄T：スコア関数 ψ で改良して **7/3-近似**

**Lemma 16**（Alexandrino et al. 2019）：任意の符号付き順列 π と short 再配置 β に対して ψ(π, β) ≤ 7/3。

**Lemma 17**（Alexandrino et al. 2019）：π ≠ ι のとき、ψ(π, β) ≥ 1 となる short 再配置 β が必ず存在する。

**Theorem 7**（Alexandrino et al. 2019）：ψ が最大の short 再配置を毎ステップ選ぶ貪欲法は、SbR̄T（λ = 3）に対して **7/3-近似**。

証明の流れ：Lemma 17 よりアルゴリズムは停止し、毎ステップ φ ≥ 1 を達成する。Lemma 16 より各操作の ψ ≤ 7/3 なので近似係数 = 7/3。

| SbT | **4/3** | Lemma 15 |
| SbRT | **2** | Theorem 1（λ−1 = 2） |
