# Sec 6: λ が大きい場合のアルゴリズム（λ = Θ(n)）

---

## 前提定義

### 順列と再配置

長さ n の符号なし／符号付き順列の定義、逆位・転置・λ-再配置の定義は Sec 3 / Sec 5 と同様。

### λ-再配置距離と無制約距離

整数 λ > 1 と再配置モデル M に対する **λ-再配置距離** cᴹλ(π) は、M の λ-再配置のみを使って π をソートする最小総コスト（f(β) = |β|）である。

λ = n のとき、長さ制約はすべての再配置を許すことと等しいため、**無制約距離** cᴹn(π) と呼ぶ。

本節では以下の記法を用いる：M ∈ {r, r̄, t, rt, r̄t}（それぞれ符号なし逆位・符号付き逆位・転置・符号なし逆位+転置・符号付き逆位+転置）。

---

## アイデア

λ 制約なしの問題（cᴹn(π) を最適化する問題）に対する近似アルゴリズムを**転用**し、その各操作を λ-再配置で置き換えることで、λ 制約付き問題への近似アルゴリズムを得る。

---

## 鍵となる補題

**Lemma 13**（λ 制約付きの最適コストは制約なしより高い）

任意の順列 π に対して cᴹλ(π) ≥ cᴹn(π)。

証明：λ 制約付き最適操作列は長さ ≤ λ ≤ n の操作のみからなるため、λ 制約なし問題の実行可能解でもある。よって無制約の最適コストはそれ以下。

---

**Lemma 14**（長い操作を λ-再配置列で模倣できる）

|β| > λ の再配置 β に対して、β と同じ効果を持つ λ-再配置のみからなる列 S が存在し、**f(S) = O(|β|²/λ)** が成り立つ（Miranda et al. 2019 の結果を利用）。

証明の概略：
- **長さ k の逆位の場合**：q = ⌈k / ⌊λ/2⌋⌉ として、q(q+1)/2 個の λ-逆位で同じ効果を再現できる。  
  総コスト f(S) ≤ λ·q²/2 = O(k²/λ)。
- **長さ k の転置の場合**：⌈(j−i)/⌊λ/2⌋⌉ × ⌈(k−j)/⌊λ/2⌋⌉ 個の λ-転置で再現できる。  
  総コスト f(S) = O(k²/λ)。

---

## 主要定理（Theorems 4–6）

x-近似アルゴリズム（λ 制約なし）を次の手順で適応する：
1. 制約なし x-近似で操作列 S（コスト f(S) ≤ x·cᴹn(π)）を得る。
2. Lemma 14 で S の各操作を λ-再配置列に置換 → 新しい列 S' のコスト f(S') ≤ (cn/λ)·f(S)。
3. Lemma 13 より cᴹn(π) ≤ cᴹλ(π) なので、近似係数は O(xn/λ)。

**Theorem 4**（逆位）：SbR または SbR̄ に対する x-近似 → λ 制約付き版に対して **O(xn/λ)-近似**

**Theorem 5**（転置）：SbT に対する x-近似 → λ 制約付き版に対して **O(xn/λ)-近似**

**Theorem 6**（逆位+転置）：SbRT または SbR̄T に対する x-近似 → λ 制約付き版に対して **O(xn/λ)-近似**

---

## 具体的な適用結果

| 問題 | 元のアルゴリズム（無制約） | λ 制約付きの近似係数 | λ = Θ(n) のとき |
|------|--------------------------|---------------------|----------------|
| SbR | Bender et al. (2008)：O(lg n) | O(n lg n / λ) | **O(lg n)** |
| SbR̄ | Swidan et al. (2004)：O(lg n) | O(n lg n / λ) | **O(lg n)** |
| SbT, SbRT, SbR̄T | Lintzmayer et al. (2018)：O(lg² n) | O(n lg² n / λ) | **O(lg² n)** |

Sec 3–5 のアルゴリズムは λ が大きいと近似係数が O(n) になるのに対し、この方法は λ = Θ(n) のとき O(lg n) / O(lg² n) を達成でき明確に優れる。
